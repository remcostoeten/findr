# Findr: Interactive File Search Tool

Findr is a powerful and user-friendly command-line tool that combines the functionality of `grep` and `find` with an interactive interface. It provides an intuitive way to search for files, folders, and content within files.

## Features

- Interactive command-line interface with user-friendly prompts
- Combined grep and find functionality
- File and content search capabilities
- Extension filtering
- Size-based filtering
- Configurable exclude patterns
- Search history management
- Rich terminal output with progress tracking
- Configuration persistence

## Installation

```bash
pip install findr
```

## Quick Start

Simply run:

```bash
findr
```

Follow the interactive prompts to configure your search.
